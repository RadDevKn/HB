# Haushaltsbuch

Budget Tracker (Python + SQLite)

## Funktionen

Einnahmen und Ausgaben eintragen mit Beschreibung und Datum
Datum wird automatisch auf korrektes Format validiert
Übersicht aller Einträge und aktueller Kontostand
Einträge nach Monat oder Jahr filtern
Auswertung höchster und durchschnittlicher EInnahme/Ausgabe
dauerhafte Speicherung in lokaler SQLite Datenbank

##Kategorien der Einnahme/Ausgabe

Miete
Lebensmittel
Transport
Freizeit
Gehalt
Sonstiges
